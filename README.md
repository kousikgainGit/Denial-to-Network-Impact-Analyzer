# Denial-to-Network Impact Analyzer

##Overview 
This project analyzes how system delays and errors impact claim processing outcomes. Inspired by telecom network performance analysis, it simulates how latency and failures affect KPIs.

##Objective
To identify the relationship between:
- System delay
- Processing time
- Error occurrence
- Claim approval/denial rates

##Tech Stack 
- Python
- Pandas

##Key KPIs Analyzed
- Approval Rate
- Denial Rate
- Average Processing Time
- Average System Delay
- Error Type Distribution

##How It Works
1. Loads claim dataset
2. Computes performance metrics (KPIs)
3. Identifies high-delay scenarios
4. Generates a summary report

## 📈 Key Insight
Higher system delays correlate with increased denial rates, similar to how network latency impacts system performance in telecom environments.

##Project Structure
- data/ → dataset
- src/ → analysis script
- output/ → generated report
